# Currently Building

## Majikah

Building trust infrastructure for the next generation of the internet.

Current projects include:

- **Majik Signature** — Digital signatures for files and text with trusted timestamping.
- **Majik Message** — End-to-end encrypted messaging with post-quantum-ready cryptography.
- **Majik Universal ID** — Self-sovereign cryptographic identity.
- **Majik Buwiz** — Offline-first invoicing with signed records.
- **Majik SLink** — Cryptographic proof of ownership for websites and public profiles.
- **Majik SDKs** — Open-source TypeScript SDKs released under Apache 2.0.
